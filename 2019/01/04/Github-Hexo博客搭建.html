使用方法

  • 目录结构

    • public 生成的网站文件，发布的站点文件。
    • source 资源文件夹，用于存放内容。
    • tag 标签文件夹。
    • archive 归档文件夹。
    • category分类文件夹。
    • downloads/code include code文件夹。
    • :lang i18n_dir 国际化文件夹。
    • _config.yml 配置文件
