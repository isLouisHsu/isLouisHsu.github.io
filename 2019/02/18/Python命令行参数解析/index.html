执行

getopt

说明

  • args=sys.argv[1:]
    传入的参数，除去sys.argv[0]，即主函数文件路径

  • shortopts='hvi:o:'
    字符串，支持形如-h的选项

    • 若无需指定参数，形如c；
    • 若必须指定参数，则需为c:；

  • longopts=['help', 'version', 'input=', 'output=']
    字符串列表，可选参数，是否支持形如--help的选项

    • 若无需指定参数，形如cmd；
    • 若必须指定参数，则需为cmd=；

执行

argsparse

说明

  • 帮助信息
    参数help，用于显示在-h帮助信息中
  • 标志位参数
    参数action='store_true'，即保存该参数为True
  • 必需参数
    置位required，即运行该程序必须带上该参数，否则报错
  • 默认参数
    参数default填写默认参数
  • 候选参数
    参数choices填写候选参数列表
