代码实现

同样的，以3维张量$X(H, W, C)$为例，指定降维顺序为$(d_1, d_2, d_3)$

  1. 在$H$维度上

    1. 将该张量转置，得到$X^{T_{d_1d_3}}(C, W, H)$
    2. 将其展开为$X_{flatten}(N_H×H)$，其中$N_H=C×W$
    3. 降维得到$X’_{flatten}(N_H×H’)$
    4. 重新将另外两维恢复，得到$X^{T_{d_1d_3}’}(C, W, H’)$
    5. 将张量转置，得到$X^{d_1}(H’, W, C)$

  2. 在$W$维度上

    1. 将$X^{d_1}(H’, W, C)$转置，得到$X^{T_{d_2d_3}}(H’, C, W)$
    2. 将其展开为$X_{flatten}(N_W×W)$，其中$N_W=H’×C$
    3. 降维得到$X’_{flatten}(N_W×W’)$
    4. 重新将另外两维恢复，得到$X^{T_{d_2d_3}’}(H’, C, W’)$
    5. 将张量转置，得到$X^{d_1d_2}(H’, W’, C)$

  3. 在$C$维度上

    1. 将$X^{d_1d_2}(H’, W’, C)$展开为$X_{flatten}(N_C×C)$，其中$N_C=H’×W’$
    2. 降维得到$X’_{flatten}(N_C×C’)$
    3. 重新将另外两维恢复，得到$X^{d_1d_2d_3}(H’, W’, C’)$
