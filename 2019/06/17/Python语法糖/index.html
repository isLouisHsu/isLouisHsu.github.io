非常pythonic的一个思想是，一切皆对象，在python中...也是有含义的

在Quora中有一条这样的回答

The other answers are correct to point out that … is the default secondary prompt in the Python interpreter, generated when you type multi-line constructs. (See 2. Using the Python Interpreter for more details.)


However, … is actually also a valid Python expression which represents the singleton Ellipsis object. This object has no methods and was originally intended only for use in advanced slicing notation in the numpy package: for example a[…, 0] selects the first elements in the last dimension of a multidimensional array. However, it is sometimes also used:

  • as a placeholder for not-yet-written code (e.g. if error: …). Code that is intentionally empty is better represented with pass.
  • when specifying type hints using the typing module (e.g. Callable[…, str] signifies a function that returns a string without specifying the call signature).
  • as a default argument value (e.g. def fn(x=…)), especially when you want to distinguish between not passing in a value and passing in None.

Since … is a singleton, you should check for it with is rather than ==.

魔法方法

Python中以__[function]__形式定义的函数就是魔术方法(Magic Function)，可以实现运算符重载等操作。常用的魔术方法有很多，如构造函数__init__和析构函数__del__、对象元素相关的如__len__, __getitem__, __setitem__, __delitem__、迭代器相关的如__iter__和__next__、运算类__add__, __sub__等等。

这类方法会在不同的时机被调用，如构造函数__init__在类的实例化时被调用，__getitem__在形如obj[key]取元素时被调用，__len__又在函数len(obj)时被调用，__call__可将实例化的对象作为可调用对象，即obj(*args)，例如

迭代器

迭代是Python访问数组元素的一种方式。迭代器是一个可以记住遍历位置的对象，从数组的一个元素开始访问，遍历到数组尾部。注意迭代器只能向前不能后退。

Python中有一个很重要的概念是可迭代对象，一般来说定义了__iter__函数的对象都具有可迭代属性，如list, set, dict, str等等，可以用内置函数isinstance进行判别

实现迭代功能需要借助以下两个函数

  • iter(object[, sentinel])：用于生成迭代器
    • object：支持迭代的集合对象；
    • sentinel：如果传递了该参数，那么object必须是可调用对象(如函数)，当迭代器返回值与sentinel一致时停止迭代，并抛出StopIteration异常。
  • next(iterator[, default])：返回迭代器的下一个项目
    • iterator：可迭代对象；
    • default：用于设置在没有下一个元素时返回该默认值。
